    base-context

    $base-context: 16;

    Description

    Set the global base context for units of measure. This also sets the default font size on the html element. This number should remain unitless.

    Used by

    em

    @function em($pixelvalue, $context) { ... }

    Description

    Convert pixel values to em.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $pixelvalue

    The desired font size.

    		Int— none
    $context

    The context / font size of the parent element.

    		Int— none

    Example

    Basic Usage

    .foo {
  font-size: em(12);
}
.bar {
  font-size: em(24, 18); // parent font size is different from $base-context, so we'll base the new context as the second param.
}

    Output

    .foo {
  font-size: 0.75em;
}
.bar {
  font-size: 1.33333em;
}

    media

    @mixin media($min, $max) { ... }

    Description

    Create min-width, max-width, or min-width and max-width media queries.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $min

    The value for min-width, can be pulled from a Neat 2.0.0 Sass map, or any other number.

    		Map or Measurement— none
    $max

    The value for max-width, can be pulled from a Neat 2.0.0 Sass map, or any other number.

    		Map or Measurement— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Basic Usage

    .foo {
  @include media(null, $tablet-portrait) {
    width: 50%;
  }
}

    Output

    @media screen and (max-width: 767px) {
  .foo {
    width: 50%;
  }
}
