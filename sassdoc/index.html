    font-legibility

    @mixin font-legibility() { ... }

    Description

    Improve the font rendering to better match client PSDs. Side effects include: slower rednering of pages. Use only when necessary!

    Parameters

    None.

    Example

    Basic Usage Sass

    .foo {
    @include font-legibility;
}

    Basic Usage CSS Output

    .foo {
      text-rendering: optimizeLegibility;
      -moz-font-feature-settings: 'liga', 'kern';
      -moz-osx-font-smoothing: grayscale;
      -webkit-font-smoothing: antialiased;
}

    wider-than

    @mixin wider-than($screen-size) { ... }

    Description

    Media Queries using Sass List Maps Define our breakpoints in $breakpoints variable and assign key value pairs in ems.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $screen-size

    Typical values are phone portrait, phone landscape, desktop, large-desktop. See available Breakpoints

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Output

    @media (min-width: 85.375em) {}

    Example

    Basic usage

    h1 {
    margin-left: 20px;

    @include wider-than(desktop) {
        margin-left: 40px;
    }

    Requires

    • [variable] breakpoints

    Author

    • Damon Cook
